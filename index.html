🇰🇷 한국어·이중언어 최강

EXAONE 시리즈는 처음부터 영어·한국어 동시 지원(Bilingual)을 핵심 설계 목표로 삼았습니다. GPT·Gemini가 한국어를 영어의 번역 언어로 처리하는 것과 달리, EXAONE은 한국어 문맥·뉘앙스를 독립적으로 이해하는 파운데이션 모델 위에서 학습되었습니다.

핵심 포인트: KoMT-Bench(한국어 MT-Bench)를 비롯한 한국어 평가 지표에서 동급 모델 최고 점수를 기록했으며, 이 한국어 기반이 수능 수학 94.5% 정답률로 직접 이어졌습니다.

📋 실사용 명령 수행력 — 7개 벤치마크 1위

EXAONE 3.5는 Arena-Hard, AlpacaEval, LiveBench, IFEval, MT-Bench, KoMT-Bench, LogicKor 등 7개 실사용 시나리오 벤치마크 전부에서 동급 최고 성능을 기록했습니다.

단순 지식 회상이 아니라 "실제로 사용자 지시를 얼마나 잘 따르는가"를 측정하는 지표들로, 산업 현장 지시 수행·문서 생성·코드 작성 등 실무 적용성과 직결됩니다.

시사점: 의료·제약 도메인처럼 정확한 지시 이행이 중요한 현장일수록, 실사용 수행력 1위라는 점이 GPT/Gemini 대비 실질적 차별점이 됩니다.

📄 장문 맥락 이해 — 4개 벤치마크 1위 · 32K 토큰

모든 EXAONE 3.5 모델은 최대 32,000 토큰의 장문 컨텍스트를 지원하며, 장문 이해 관련 4개 벤치마크에서 동급 모델 최고 성능을 달성했습니다.

이는 긴 임상 시험 보고서, 의약품 설명서, 연구 논문 전문을 한 번에 처리할 수 있음을 의미합니다. 문서를 잘라 여러 번 질의해야 하는 다른 모델 대비 바이오헬스케어 데이터 분석에 구조적으로 유리합니다.

🧮 추론·수학·코딩 특화 — EXAONE Deep 32B

EXAONE Deep은 EXAONE 3.5를 기반으로 추론(Reasoning) 능력에 집중 파인튜닝한 모델로, 2025년 3월 NVIDIA GTC에서 오픈소스로 공개되었습니다. 국내 기업 최초의 추론 특화 AI입니다.

💡 압도적 학습 효율 — Qwen 2.5 대비 연산량 1/2.77

EXAONE 3.5 32B는 동급 최고 수준의 성능을 가장 낮은 학습 비용으로 달성한다는 점이 특징입니다. 기술 보고서에 따르면, 동급 모델인 Qwen 2.5 32B가 EXAONE 3.5 32B보다 2.77배 많은 연산량을 필요로 합니다.

실용적 의미: 동등 성능을 더 적은 GPU·전력으로 구동할 수 있다는 뜻입니다. 온프레미스 서버 배포, 클라우드 API 비용 모두에서 경쟁 모델 대비 경제적입니다. LG CNS 환경에서 배포·운영 비용 최적화에 유리합니다.

📦 경량화 라인업 — 현장 배포 유연성

EXAONE Deep은 단일 모델이 아니라 세 가지 크기의 라인업으로 제공됩니다. 병원 내부 온프레미스부터 모바일 온디바이스까지 환경에 맞게 선택할 수 있습니다.

모델 파라미터 32B 대비 성능 주요 용도
EXAONE Deep 32B 320억 100% (기준) 서버·클라우드 고성능 추론
EXAONE Deep 7.8B 78억 (24% 크기) 95% 유지 온프레미스 경량 서버
EXAONE Deep 2.4B 24억 (7.5% 크기) 86% 유지 온디바이스·엣지 디바이스

7.8B 모델은 OpenAI o1-mini보다 우수한 성능을 보이며, 상용 추론 API 없이도 자체 인프라에서 구동 가능합니다.

🔜 명확한 업그레이드 로드맵

LG AI연구원은 지속적인 모델 업그레이드를 공개적으로 예고하고 있습니다. EXAONE 3.5 → EXAONE Deep → EXAONE 4.0 순으로 이미 공개가 진행 중이며, MoE(Mixture of Experts) 아키텍처를 적용한 K-EXAONE 236B도 출시되어 있습니다.

협력 관점: 지금 차의과학대학교와 구축하는 협력 프레임워크는 단발성이 아니라, EXAONE의 버전업마다 지속적으로 강화되는 장기 파트너십의 기반이 됩니다.
